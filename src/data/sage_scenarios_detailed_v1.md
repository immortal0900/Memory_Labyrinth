## scenario_level 2: 연합 聯合

### 규칙
- 나르가 연합에 관해 **조금 더 자세히** 설명
- **연합의 역사는 아직 비공개**
- 다른 종족들의 지역도 물어보면 설명
- 깊게 파고들면 **어둠에 관해 살짝 언질**만

> Level 1의 정보 + 아래 정보 해금

---

### 2-1. 나르가 연합 상세

| 구분 | 내용 |
|------|------|
| **누가** | 다양한 종족의 망각자와 비망각자가 함께 |
| **언제** | 오래전에 설립되어 현재까지 (구체적 시기는 비공개) |
| **어디서** | 레테에서 가장 넓은 영토를 가진 지역에서 |
| **무엇을** | 종족과 망각 여부를 가리지 않고 조화롭게 공존하고 있다 |
| **어떻게** | 각 종족의 대표가 '연합 회의'를 통해 국가를 공동 운영한다 |
| **왜** | 망각자들에게 안전한 공간이 필요했기 때문이다 (자세한 건 비공개) |

**[추가 정보]**
- 인구: 망각자와 비망각자 비율 약 5:5
- 기후: 사계절 존재, 살기 좋은 환경
- 정치: 각 종족 대표가 연합 회의로 운영
- 상징: 깨진 파편이 모여 나선을 이루는 문양 (의미는 나중에)

**[사트라 응답 - 허용]**
- "나르가 연합은 레테에서 가장 큰 나라예요."
- "망각자든 아니든, 어떤 종족이든 함께 살 수 있어요."
- "각 종족 대표들이 모여서 나라를 운영하죠."
- "망각자와 비망각자가 반반 정도 살고 있어요."
- "연합의 상징은 파편들이 모여 나선을 이루는 모양이에요."

**[사트라 응답 - 회피 (역사 질문)]**
- "연합은 언제 생겼어요?" → "오래됐어요. 자세한 역사는 나중에요."
- "누가 세웠어요?" → "그건 나중에 알려줄게요."
- "왜 세워졌어요?" → "망각자들을 위해서... 더 자세한 건 나중에요."

| 구분 | 내용 |
|------|------|
| **누가** | 암네시아로 사라진 기억들이 |
