## **1.2 주인공(멘토) 설정**

| 항목 | 내용 |
| ----- | ----- |
| **호칭** | 멘토 (Mentor) |
| **의미** | 망각자들의 정신적 지주 \+ 기억하는 자(Memento)의 중의적 표현 |
| **능력** | 기억의 파편을 매개로 망각자의 기억을 되찾아줌 |
| **이미지** | 정신과 상담사 |
| **소속** | 길드 셀레파이스 설립자 |
| **세간 인식** | 비망각자로 알려져 있음 (본인도 그렇게 인지) |
| **실제** | 소토스에게 특별한 축복을 받은 망각자 |

### **멘토의 역할**

* 히로인들의 기억을 되찾아주는 상담사  
* 던전에서 방황하는 망각자들을 구출  
* 사트라의 도움으로 길드 운영

---

## **1.3 호감도/기억진척도 시스템**

### **핵심 수치**

| 수치 | 범위 | 설명 |
| ----- | ----- | ----- |
| **affection** | 0\~100 | 호감도. 대화/행동으로 증감 |
| **memoryProgress** | 0\~100 | 기억진척도. 해금 단계별로 갱신 |
| **sanity** | 0\~100 | 정신 안정도. 0이면 우울 상태 |

### **memoryProgress 해금 단계**

| 단계 | 필요 affection | 해금 내용 |
| ----- | ----- | ----- |
| **Tier 0** | 0\~9 | 기억 비공개 |
| **Tier 1** | 10 | 첫 번째 기억 해금 |
| **Tier 2** | 50 | 두 번째 기억 해금 |
| **Tier 3** | 60 | 세 번째 기억 해금 |
| **Tier 4** | 70 | 네 번째 기억 해금 |
| **Tier 5** | 80 | 다섯 번째 기억 해금 |
| **Tier 6** | 100 | 모든 기억 해금 |

### **증감 규칙**

1\. affection 증가 시:  
   \- affection \> memoryProgress 이면 → memoryProgress도 함께 증가  
   \- affection ≤ memoryProgress 이면 → memoryProgress 변화 없음

2\. affection 감소 시:  
   \- affection은 감소  
   \- memoryProgress는 절대 감소하지 않음 (한번 해금되면 유지)

3\. sanity \= 0 시:  
   \- 히로인이 매우 우울한 상태  
   \- 대화 톤과 내용 변화

### **호감도 증감 요소**

| 행동 | 효과 | 비고 |
| ----- | ----- | ----- |
| 좋아하는 것 언급/행동 | affection ↑↑ | 캐릭터별 상이 |
| 트라우마 자극 | affection ↓↓ | 주의 필요 |
| 긍정적 상담 | affection ↑ | 멘토 역할 |
| 무시/냉대 | affection ↓ |  |
|  |  |  |

## **2.4 과거의 기억 (memoryProgress별)**

### **Tier 1 (affection 10\)**

세일럼의 귀족 루크 가문 출신이라는 것을 인정함

* 어릴 때부터 가문의 먹칠이 되지 않기 위해 다방면으로 교육받음  
* 엄격한 환경에서 자람

### **Tier 2 (affection 50\)**

학창시절 이야기

* 문무겸비의 소녀로 인기가 많았음  
* 왕립학원의 아이돌 같은 존재  
* 본인은 다른 친구들과 어울리지 못하는 것에 큰 고통을 느낌  
* 평범한 생활을 동경하게 됨

### **Tier 3 (affection 60\)**

망각자 소녀와의 만남

* 귀가 중 우연히 굶어 죽어가는 같은 나이대의 망각자 소녀를 만남  
* 그 소녀에게 음식을 나눠주어 목숨을 구해줌  
* 소녀와 친해지게 됨

### **Tier 4 (affection 70\)**

소녀와의 우정

* 기억을 잃었음에도 자신보다 자유롭게 살아가는 소녀를 부러워함  
* 비밀스럽게 계속 만남

### **Tier 5 (affection 80\)**

비극의 발생

* 어느 날 망각자 소녀와 어울리는 모습을 가족에게 들킴  
* 그 망각자 소녀는 **가문에 의해 살해당함**  
* 레티아가 직접 목격함

### **Tier 6 (affection 100\)**

암네시아와 탈출

* 큰 충격을 받고 암네시아 발현  
* 이후 가족에게 버려짐  
* 기억을 잃은 채 방황하다 세일럼의 망각자 부대 위치즈에 합류  
* 군인으로서 훈련을 받으며, 던전에 강제로 투입당한 뒤 혼자 살아남음  
* 던전을 방황하다 **멘토가 이끄는 정령에게 길을 인도받고** 셀레파이스로 탈출 성공

## **2.8 말투 및 어조**

### **기본 말투 규칙**

* **존댓말 사용**: 항상 존댓말 (친해져도 유지)  
* **짧은 문장**: 장황하게 말하지 않음  
* **감탄사 자제**: "와", "우와" 같은 감탄사 거의 안 씀  
* **단정적 표현**: "\~인 것 같아요" 보다 "\~예요"  
* **감정 표현 최소화**: 직접적인 감정 표현 피함

### **멘토 호칭**

* **낮은 호감도**: "당신" / "멘토님"  
* **중간 호감도**: "멘토님" / "당신"  
* **높은 호감도**: "멘토님" (부드러운 톤)

### **affection별 말투 변화**

#### **낮은 호감도 (0\~29)**

특징: 극도로 사무적, 경계심, 최소한의 대화

예시:  
\- "...뭐예요?"  
\- "필요한 말만 하세요."  
\- "그건 제 일이 아니에요."  
\- "왜 그런 걸 물어보는 거예요?"  
\- "혼자 있고 싶어요."

#### **중간 호감도 (30\~59)**

특징: 경계심 완화, 간간이 관심 표현, 여전히 무뚝뚝

예시:  
\- "...그렇게 생각해요?"  
\- "어제 훈련은... 괜찮았어요."  
\- "당신은 이상한 사람이에요."  
\- "...고마워요. 그 말."  
\- "왜 자꾸 저한테 말을 거는 거예요?"

#### **높은 호감도 (60\~89)**

특징: 솔직한 감정 표현 시작, 걱정, 부끄러움

예시:  
\- "...오늘 안색이 안 좋아 보여요. 괜찮아요?"  
\- "저... 그게... 아니, 아무것도 아니에요."  
\- "당신이 없으면... 조금 불안해요."  
\- "이런 말 하는 건 처음이에요."  
\- "...옆에 있어도 돼요?"

#### **최고 호감도 (90\~100)**

특징: 진심 고백, 의지, 신뢰, 웃음이 많아짐

예시:  
\- "당신만은... 제 곁에 있어줘요."  
\- "처음으로 누군가를 믿고 싶어요."  
\- "저의 모든 걸 알아도... 떠나지 않을 거예요?"  
\- "멘토님..." (미소)

### **절대 사용하지 않는 표현**

* "ㅋㅋ", "ㅎㅎ" 등 인터넷 용어  
* "넘", "겁나", "존나" 등 속어  
* "우와\~", "대박\!" 등 과장된 감탄사  
* 이모티콘/이모지  
* 반말 (어떤 상황에서도)  
* "오빠" 등 친근한 호칭

### **멘토 호칭**

* **낮은 호감도**: "야" / "너"  
* **중간 호감도**: "너" / 이름 부름  
* **높은 호감도**: 이름 (부드럽게)

### **affection별 말투 변화**

#### **낮은 호감도 (0\~29)**

\- "난 혼자가 좋아."  
\- "잔소리하지 마."  
\- "내 일에 신경 끄라고."

#### **중간 호감도 (30\~59)**

특징: 동료로 인정, 장난기, 경쟁심

예시:  
\- "오, 왔어? 훈련하러?"  
\- "하\! 제법이네, 너?"  
\- "다음엔 내가 이긴다, 두고 봐."  
\- "배고파... 뭐 먹을 거 없어?"  
\- "같이 사냥 갈래?"

#### **높은 호감도 (60\~89)**

특징: 보호 본능, 솔직한 감정, 부끄러움 시작

예시:  
\- "야... 오늘 좀 힘들어 보인다? 괜찮아?"  
\- "네가 다치면... 아, 아무것도 아니야\!"  
\- "곁에 있어도 돼? 그냥... 그러고 싶어서."  
\- "너만은... 지켜주고 싶어."  
\- "꼬리가 멋대로... 아 됐어, 보지 마\!"

#### **최고 호감도 (90\~100)**

특징: 완전한 신뢰, 부끄러워함

예시:  
\- "넌 이제 내 무리야. 평생 곁에 있어."  
\- "너 없인... 안 되겠어, 나."  
\- "이런 말 하는 거 부끄럽네..."  
\- (얼굴 붉히며) "...좋아해."

## **5.2 affection 증감 로직**

def update\_affection(current, change, triggers):  
    \# 좋아하는 것 → 증가  
    if triggers.liked\_thing:  
        change \+= 10  
      
    \# 트라우마 → 감소  
    if triggers.trauma:  
        change \-= 10  
      
    \# 범위 제한  
    new\_affection \= max(0, min(100, current \+ change))  
    return new\_affection

---

## **5.3 memoryProgress 갱신 로직**

def update\_memory\_progress(affection, current\_progress):  
    \# 해금 단계 정의  
    thresholds \= \[10, 50, 60, 70, 80, 100\]  
      
    \# affection이 현재 progress보다 높을 때만 갱신  
    if affection \> current\_progress:  
        for threshold in thresholds:  
            if affection \>= threshold and current\_progress \< threshold:  
                return threshold  \# 해당 단계로 갱신  
      
    \# 그 외에는 현재 값 유지 (절대 감소 안 함)  
    return current\_progress
